# Internship Tracker

A lightweight, dark-themed internship application tracker that syncs to the cloud in real time.

## Features

- Add applications manually or by pasting a job posting URL
- Tracks company, role, location, type, status, priority, salary, and notes
- Real-time sync via Firebase Realtime Database
- Dashboard with status breakdown and key stats
- Export to CSV, XLSX, or copy rows for pasting into a spreadsheet

## File Structure

```
├── index.html   # HTML structure
├── style.css    # All styling
├── app.js       # App logic + Firebase integration
└── README.md
```

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- Firebase Realtime Database (cloud sync)
- SheetJS (XLSX export)
- Hosted on GitHub Pages
